# Personal website

This personal website was created using React and has some external APIs developped by me in .NET connected to it's frontend.

## Prerequisites
React npm
axios
react-intersection-observer

## Local launch
Open the root folder with VScode or an IDE of your choice and run: npm start to run the website locally

## Features
This is a single page website which contains all of the necessary information.

1. About section: this section contains basic information about me with expandable boxes.
2. Projects section: this section displays my projects and more projects can be dynamically added via the function inside.
3. API section: this section has my APIs connected to it which allows the user to use them live.
4. Contact section: this section contains a Formspree api link which is used to send me an email with the text in the form.
